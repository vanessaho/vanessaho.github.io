 BuzzWord

JavaFX, Java, CSS, XML, JSON

BuzzWord is a word game that requires the player to find as many words as they can on the scrambled board of letters and beat the required score before the timer ends to move on to the next round.

BuzzWord is a JavaFX application that I programmed in my sophomore year of college. It utilizes a recursive acyclic algorithm to travel through the board and find all potential words longer than three letters. This method ensures that there would be enough words to beat the required score and if not, the board will rearrange new letters until it does.

The user's credentials and progress are saved into a "secure" JSON file and are loaded back into the game if those same credentials are inputted.

 Centigrade

HTML5, CSS3, JavaScript/jQuery, Spring Boot Framework, Thymeleaf Templating Engine, MySQL

Centigrade is a web application that serves as a Rotten Tomatoes replica, except better.

It utilized a remote MySQL database server provided by Stony Brook University to store all of our movie/TV show information and user authentication credentials.

By using Thymeleaf's template engine, the web application can dynamically create web pages by pulling data from Spring Boot's back-end code, removing the need to create each page individually.

Centigrade is my senior design project; it has massive functionality including, but not limited to, viewing movie/TV show information, applying to become a critic and having special critic controls, as well as commenting and rating movies/TV shows.

 Coracle

HTML5, CSS3, JavaScript/jQuery, Bootstrap, Python Flask, MySQL

Coracle is a web application made with Flask, which is a Python microframework.

It pulls its information from a MySQL database and allows the user to plan trips using built-in attractions.

This application supports registrations/logins, AJAX calls to the back-end for autocompleting fields, and SQL queries to insert/fetch data on prices and locations.
