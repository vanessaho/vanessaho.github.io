Smart and Inclusive

My first real JavaFX project, BuzzWord takes a game board full of scrambled letters and encourages the player to create words out of them.

A requirement for this game was to ensure that there are enough words to finish the current level.

I accomplished this essential guideline by including a recursive acyclic algorithm that searches the entire board for possible words and refreshes it if that prerequisite wasn't met.

Stony Brook Innovation

My first assignment at Stony Brook Research was to create a new but separate section called Stony Brook Innovation.

The point of the page is to highlight each category that Stony Brook University is currently working with in terms of research and technology.

I used CSS to create the separate blocks for categories and PHP to access data from the website's SQL tables. Moreover, since the base data type is XML, I had to utilize Drupal's xPathParser module to create separate pages for each technology.
